# Strength Program Calculator

## Описание
Интерактивный и полностью клиентский генератор силовых программ на 6+ недель. Интерфейс вдохновлён Apple-like эстетикой и остаётся отзывчивым и быстрым даже без подключения к серверу.

## Возможности
- ввод текущих рабочих показателей (вес, повторения, тип движения, цель, длительность)
- выбор схемы прогрессии: Top Set, Linear 5x5, Double Progression
- автоматическая генерация недельных планов с топ-сетами и дополнительными подходами
- сохранение готовых программ в `localStorage`
- загрузка, удаление и моментальное переключение между программами
- чистый и доступный UI с адаптацией под мобильные устройства
- автономная работа в браузере без бэкенда и внешних API

## Развёртывание на GitHub Pages
1. Создайте публичный репозиторий на GitHub и загрузите содержимое проекта.
2. Убедитесь, что `index.html` находится в корне репозитория (рядом со `styles/` и `js/`).
3. Откройте настройки репозитория → **Pages** → выберите "Deploy from branch" и ветку `main` (или нужную).
4. После публикации GitHub выдаст постоянную ссылку вида `https://<user>.github.io/<repo>/`.

## Структура проекта
```
strength-program-calculator/
├── index.html          # точка входа и разметка приложения
├── styles/base.css     # глобальные стили и адаптивная сетка
├── js/main.js          # инициализация формы и генерации программ
├── js/ui/              # визуальные компоненты (programView, programList, form)
├── js/calculators/     # схемы прогрессии и расчётные модули
├── js/storage.js       # обёртки над localStorage
├── README.md
└── STYLEGUIDE.md
```

## Технологии
- **HTML5** — структурированная разметка и ARIA-атрибуты
- **CSS3** — адаптивный Apple-like UI без сторонних фреймворков
- **ES Modules** — модульная архитектура JavaScript с чистыми калькуляторами
